1. What is oops?
   oops is a programing pradigm which uses objects that contains data and functions to
   operate on the data together.

2. Features of oops?
   i.  Abstraction
   ii. Encapsulation
   iii.Inheritance
   iv. Polymorphism

3. Advantages?
   i.Reuse of code leads to faster development and maintainbilty.
   ii.Extensible property of objects makes it easier to implement changes.
   iii. Lower cost of development.

4. Disadvantages?
   i. Larger program size.
   ii. Slower Program than functional.
